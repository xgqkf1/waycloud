## Por que o TTFB importa mais do que parece

Antes de falar de preço, vale entender por que a velocidade do servidor não é frescura. O TTFB (Time to First Byte) é o tempo que passa entre o usuário clicar no link do seu site e o servidor começar a responder. 

16 ms é o número que a Way Cloud apresenta nos testes. Para ter uma ideia, a maioria das hospedagens compartilhadas bate na casa dos 400–800 ms. Essa diferença vai direto no Google: o Core Web Vitals pondera o TTFB e, quanto mais rápido o servidor, melhor o ranqueamento — e menos visitante abandona a página antes dela carregar.
